# 🎂 Interactive Birthday Wish Website

A vibrant, interactive web application designed to deliver a personalized birthday experience. Built with a focus on smooth animations, audio integration, and user engagement.

## ✨ Features
* **Surprise Reveal:** Interactive "Tap to open" gift box animation.
* **Balloon & Confetti Effects:** Dynamic visual celebrations upon opening.
* **Interactive Cake:** "Blow out" the candles by clicking them (includes a counter and "Relight" functionality).
* **Audio Experience:** Integrated "Birthday Serenade" music player with play/pause controls.
* **Personalized Wishes:** Aesthetic card layout with rotating birthday messages.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and glassmorphism effects.
* **JavaScript (ES6):** Logic for candle interaction, music toggle, and event-driven animations.
